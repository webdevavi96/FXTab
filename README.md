# FXTab

A Chrome extension for enhanced tabbed browsing and workspace management.

## Features

- **Tab Organization** - Efficiently manage and organize multiple open tabs
- **Quick Navigation** - Quickly switch between open tabs and workspaces
- **Tab Grouping** - Group related tabs together for better organization
- **Custom Shortcuts** - Customizable keyboard shortcuts for common actions

## Installation

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer Mode** (toggle in top right)
3. Click **Load unpacked**
4. Select the FXTab extension folder
5. The extension will appear in your extensions list

## Usage

- Click the FXTab icon in the toolbar to access features
- Right-click on tabs for quick actions
- Use keyboard shortcuts to navigate between tabs
